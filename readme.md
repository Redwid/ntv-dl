###App for downloading ntv tv series

####Install pyobdc
For MacOS:
``brew install unixodbc``

For Debian:
``sudo apt install unixodbc-dev``

####Run app
``python3 ntv-dl.py -u your_db_user_name -p your_db_user_password``
